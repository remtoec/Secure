# 依附地形圖 · Attachment Topography（讀書會版）

一個以繁體中文（香港）重製的依附風格測驗網頁應用，供讀書會使用。
計分方法依據 Amir Levine 醫生著作《Secure》(2026)：

- 就**某一段特定關係**回答 9 條題目（1–7 分 Likert 量表）
- **迴避分數** = 第 1–6 題平均（第 1–4 題反向計分）
- **焦慮分數** = 第 7–9 題平均
- 以 4 分為分界，繪畫在四象限「依附地形圖」上：
  安全型（低焦慮低迴避）、焦慮型、迴避型、恐懼迴避型

題目文字採用可自由使用的研究量表 ECR-RS（Fraley, Heffernan, Vicary & Brumbaugh, 2011）之中文改寫版。

## 功能

- **多對象**：伴侶、父母、朋友、同事⋯⋯每位對象一種顏色，全部繪在同一張地形圖上
- **時間軌跡**：同一對象可重複測驗，地圖上以連線顯示關係隨時間的移動
- **無需登入**：所有紀錄只存在裝置的瀏覽器（localStorage）
- **匯出／匯入**：一鍵匯出 JSON 備份檔，可換機還原或與人分享（支援合併或取代）
- **關係檢核表**：附《Secure》書末 15 題「這段關係值得投入嗎？」檢核（CARRP / SIMIs）
- 深／淺色主題、手機優先設計

## 使用方法

純靜態單一檔案，無需安裝任何東西：

1. 直接用瀏覽器開啟 `index.html`；或
2. 開啟 GitHub Pages（Settings → Pages → 選擇分支），把網址分享給讀書會朋友。

> 本工具僅供教育、自我了解及讀書會討論之用，並非臨床診斷。
